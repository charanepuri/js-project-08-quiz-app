# Quiz App 🧠

## 📌 Project Overview

This project is an interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.
It presents multiple-choice questions, allows users to navigate between them, and displays the final score at the end.

The app supports **Next / Previous navigation**, remembers selected answers, and shows progress through the quiz.

This project is part of my **JavaScript Practical Projects Series 🚀**, where I build real-world applications to strengthen my frontend development skills.

---

## 🎯 Features

- Displays **multiple-choice questions**
- Navigate using:
  - Next button
  - Previous button

- Shows **question number (progress indicator)**
- Remembers user-selected answers
- Displays final score after completion
- Clean and responsive UI

## ⚙️ How It Works

1. Questions are stored in an array of objects.
2. Each question contains:
   - Question text
   - Answer options
   - Correct answer index

3. When the app loads:
   - First question is displayed
   - Progress indicator is shown

4. User can:
   - Select an answer
   - Navigate between questions

5. Selected answers are stored in an array.
6. On completion:
   - Score is calculated
   - Result is displayed
